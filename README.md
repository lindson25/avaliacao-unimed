## Stack(s) utilizada(s) e justificativa

A implementação dos katas foi realizada utilizando:

**Backend**
- Java
- Spring Boot
- Maven
- PostgreSQL
- JUnit

**Justificativa da escolha**

A escolha da stack Java + Spring Boot foi motivada por ser o ecossistema no qual possuo maior domínio atualmente no desenvolvimento de APIs REST e aplicações backend.

Apesar da stack sugerida na avaliação incluir C#/.NET, os conceitos de engenharia aplicados são equivalentes entre os ecossistemas, permitindo demonstrar capacidade técnica independente da linguagem utilizada.
